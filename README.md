# Image Gallery - Галерея изображений

Приложение для добавления изображений по URL с валидацией.

## Функциональность

- Добавление изображений по URL и названию
- Валидация URL (проверка, что по ссылке действительно изображение)
- Добавление по кнопке или по клавише Enter
- Очистка полей после успешного добавления
- Сообщение об ошибке при неверном URL
- Удаление изображений из галереи
- Адаптивная сетка изображений

## Технологии

- JavaScript (ES6)
- Webpack
- HTML5/CSS3
- GitHub Actions (CI/CD)

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/AleksandrMuzhev/ahj-image-gallery.git

# Установка зависимостей
cd ahj-image-gallery
yarn install

# Запуск в режиме разработки
yarn start

# Сборка production версии
yarn build
```
